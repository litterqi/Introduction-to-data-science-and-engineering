### 第4题：

我们首先定义了矩阵 A ，最大迭代次数 max_iter 和收敛阈值 tol。然后我们使用随机向量初始化 x，并通过 for 循环开始执行编程幂迭代法。在每个迭代步骤中，我们计算 y = Ax 和特征值 lmd，并计算误差 err。如果误差小于指定的收敛阈值 tol，则结束循环。否则更新 x，继续迭代。最后，我们输出计算出的最大特征值 lmd。
